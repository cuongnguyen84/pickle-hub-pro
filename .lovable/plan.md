# Highlight nickname trong chat livestream (Admin/Moderator)

## Ý tưởng gợi ý

Có 3 cách tiếp cận, từ đơn giản đến phức tạp:

### Option A: VIP Badge + Màu nổi bật

- Admin có thể đánh dấu user là "VIP" cho 1 livestream cụ thể
- Nickname VIP hiển thị màu vàng/gold thay vì màu mặc định
- Có badge ngôi sao ⭐ bên cạnh tên
- Đơn giản, dễ nhận biết

### Option B: Custom highlight với nhiều kiểu

- Admin chọn từ vài preset: VIP (vàng), Sponsor (xanh), Special Guest (tím)
- Mỗi kiểu có màu nickname + badge icon riêng
- Linh hoạt hơn Option A

### Option C: Tự chọn màu + text tùy ý

- Admin pick màu bất kỳ + nhập label tùy ý (VD: "Nhà tài trợ", "Khách mời")
- Phức tạp nhất, nhiều UI nhất

## Triển khai Option B
