# ThePickleHub — Native iOS (SwiftUI)

Full-native SwiftUI rewrite of ThePickleHub. **Reuses the existing Supabase
backend unchanged** — only the client layer is native. The Capacitor wrapper in
`../ios/` stays the shipping app until this reaches feature parity.

- Min target: **iOS 17**
- Dev bundle id: `net.thepicklehub.app.dev` (kept distinct from the live
  `net.thepicklehub.app` so both can coexist on a device / App Store)
- Project is generated by **XcodeGen** from `project.yml` — the `.xcodeproj` is
  git-ignored. Edit `project.yml`, not the project in Xcode.

## Setup

```sh
brew install xcodegen          # one-time
cd apple
cp Config/Secrets.example.xcconfig Config/Secrets.xcconfig   # then fill in values
xcodegen generate
open ThePickleHub.xcodeproj
```

Build/run from CLI. **DerivedData + SPM checkouts are kept on the external
volume** to spare the main disk (which is tight):

```sh
xcodegen generate
xcodebuild -scheme ThePickleHub \
  -destination 'platform=iOS Simulator,name=iPhone 17' \
  -derivedDataPath /Volumes/CMBackup/picklehub-ios/DerivedData build
```

> The iOS Simulator runtime (~8GB) lives on the boot volume under
> `/Library/Developer/CoreSimulator` — Apple does not support relocating it.
> Only build output is offloaded.

## What Phase 0 ships

A scaffold that **de-risks the migration** before any feature work:

| Piece | Proves |
|-------|--------|
| `SupabaseManager` + `AppConfig` | Client wiring from `Secrets.xcconfig` → Info.plist → runtime |
| `LoginView` → email/password sign-in | supabase-swift handles **ES256** auth (the web gateway bug does not apply natively) |
| "Test public edge function" (`geo-check`) | Network + Functions gateway reachable |
| "Test authed read" (`profiles` self-row) | User **JWT + RLS** work end-to-end via PostgREST |
| `TLColor` / `TLComponents` | The Line design tokens ported to SwiftUI |

If both probes pass with a real account, the architecture is validated.

## Layout

```
apple/
├── project.yml                 # XcodeGen spec (source of truth)
├── Config/                     # Secrets.xcconfig (git-ignored) + example
├── ThePickleHub/
│   ├── App/                    # @main entry + Info.plist
│   ├── Core/
│   │   ├── Supabase/           # client + config
│   │   ├── Auth/               # SessionStore (@Observable)
│   │   └── Networking/         # BackendProbe
│   ├── DesignSystem/           # TLColor, TLRadius, components
│   └── Features/               # Root, Auth, Home (placeholder)
└── Tests/
```
