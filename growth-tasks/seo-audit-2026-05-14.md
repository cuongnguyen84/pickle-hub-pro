## 2. Site-wide audit

### 2.1 Sitemap

`/sitemap.xml` index trỏ 7 segment:
- `/sitemap-static.xml` — landing/tools/privacy/terms
- `/sitemap-blog.xml` — EN + VI blog (with hreflang xhtml:link)
- `/sitemap-tournaments.xml`
- `/sitemap-matches.xml`
- `/sitemap-events.xml` — events + clubs (✅ PR69, real prod data: 3 events + 3 clubs)
- `/sitemap-players.xml`
- `/sitemap-venues.xml` (page chưa tồn tại — emit URL để Google làm nóng pipeline)

✅ `sitemap-events.xml` emit hreflang vi/en/x-default đầy đủ.
⚠ **NHƯNG** prerender HTML của `/social/:slug` và `/clb/:slug` KHÔNG echo
hreflang trở lại — bị bất đối xứng (sitemap signal vs page signal).
Sẽ liệt kê là I-5.

### 2.2 robots.txt (`public/robots.txt`)

Đang allow/disallow theo legacy paths cũ:
- ❌ KHÔNG có Allow rule cho `/social/`, `/social`, `/clb/`, `/feed`, `/nguoi-choi/`
  (User-agent `*` mặc định allow nhưng dòng Allow explicit nên đồng bộ).
- ❌ KHÔNG có Disallow cho `/social/:slug/danh-sach`, `/clb/:slug/quan-ly`,
  `/dang-ky/`, `/khoi-phuc-dang-ky`, `/clubs/new`, `/clb/:slug/social/moi`,
  `/clb/:slug/quan-ly/*`.
- Disallow `/matches/` đang là legacy path (App.tsx có `/matches/:matchId/score`
  — that's correct, private). OK.
- Disallow `/embed/`, `/admin`, `/creator`, `/login`, `/auth/`, `/notifications` ✓.
- ⚠ Disallow `/join/` — đúng (invite link).
- Vẫn đang `Disallow: /notifications` nhưng `renderNotificationsShell`
  đã noindex; redundant nhưng không sai.

### 2.3 URL structure consistency

- ✅ Sau PR69 toàn bộ social paths đã chuẩn về `/social/`, `/clb/`. Cả
  `_redirects` (server 301) lẫn SPA Navigate alias (client 301-equivalent)
  đều bắt `/su-kien/*` legacy.
- ⚠ `/clubs` (list, English-ish) vs `/clb/:slug` (detail, Vietnamese)
  bất đối xứng. CLAUDE.md đã có nhưng audience VN — list page nên là
  `/clb` (no `s`) để đồng nhất, hoặc cả hai phải redirect ra một canonical.
- ⚠ **`/u/:slug` vs `/nguoi-choi/:username`** — 2 trang khác nhau (2 page
  components: `PublicProfile` vs `PlayerProfile`). Cần điều tra: đây là
  bug double-page hay 2 surface khác nhau cố ý? Bots hiện chỉ index
  `/nguoi-choi/` (có prerender), `/u/` rơi vào renderDefault.

### 2.4 Internal linking + breadcrumb

- ✅ Homepage nav link sang tournaments/tools/blog/news/videos/forum
  với anchor text mô tả.
- ⚠ `renderSocialEvent` breadcrumb có label "Sự kiện CLB" trỏ về
  `/clb/{slug}` — label sai (phải là tên CLB chứ không phải generic
  "Sự kiện CLB"). Đây là Codex-level nit nhưng ảnh hưởng signal hierarchy.
- ⚠ Homepage prerender body không link tới `/social`, `/clubs`, `/feed`
  — 3 surface mới sau Sprint 4+5 không có internal link từ home cho
  bot crawl. Crawl budget waste.
- ⚠ `/clb/:slug` không link đến `/social/:slug/danh-sach` từ event card
  trong prerender (only HTML server-side); nhưng đây là tính năng
  organizer-only nên không cần.

### 2.5 404 page

`render404` (functions/_lib/render/index.ts:912) trả `status: 404` +
title "404 — Không tìm thấy trang | ThePickleHub" + canonical đến URL
yêu cầu. ✅ Đúng — không bị soft-404.

`/tran-dau/abc-fake` → 404 chuẩn (xác nhận qua curl).
`/clb/cuc-pickleball` → 404 chuẩn (slug không tồn tại).

⚠ 404 page KHÔNG có `<meta name="robots" content="noindex">`. Nếu Google
gặp 404 nó tự loại — nhưng thêm noindex là defense-in-depth standard.

## 3. Technical SEO

### 3.1 Core Web Vitals

- Bot path KV cache HIT (`X-Prerender-Cache: HIT`) sau lần render đầu,
  TTL 6h. Cold cache ~1s (Tokyo Supabase RTT). Đã tốt.
- User path SPA bundle. `index.html` đã có:
  - Inline critical CSS ✓
  - Preconnect Supabase + fonts.gstatic ✓
  - Preload 3 critical woff2 (Geist, GeistMono, InstrumentSerif) ✓
  - GA defer via `requestIdleCallback` ✓
- ⚠ KHÔNG có `<link rel="preload" as="image">` cho LCP image của detail
  pages (vì client-side fetch); chấp nhận được nhưng để ý.
- ⚠ Mux player + livestream embed có thể là LCP element cho live
  pages, không preload.

### 3.2 Image optimization

- ❌ Bot HTML KHÔNG include `<img>` tags (chỉ có text/links/JSON-LD).
  Schema `image` field chỉ set ở VI blog. EN blog `BlogPosting` thiếu
  `image` (I-10).
- ❌ Event detail prerender không inject event cover image (nếu có); chỉ
  dùng default og-image.png. Mất cơ hội rich card cho Facebook/Twitter share.
- ❌ Club detail prerender same — không inject `logo_url` vào og:image.

### 3.3 Mobile responsive

- `TheLineLayout` đã chuẩn mobile-first (CLAUDE memory note). ✓
- Static index.html có viewport meta đầy đủ. ✓

### 3.4 HTTP headers

- `public/_headers`:
  - Static assets `Cache-Control: public, max-age=31536000, immutable` ✓
  - HTML root `max-age=300, s-maxage=3600` ✓
  - Sitemap `max-age=300` ✓
  - X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy ✓
- Bot HTML response từ middleware: `Cache-Control: no-store` cho HIT,
  `public, max-age=3600` cho MISS. ✓

### 3.5 Prerender legacy worker

CLAUDE.md note: legacy `prerender-worker` CF Worker chạy parallel.
Phase 1 chưa verify — Phase 2 cần `wrangler tail` để xác nhận traffic
share. Nếu CF Pages middleware đã serve toàn bộ bot traffic thì có thể
retire worker; nếu còn cũ thì cả 2 phải đồng bộ patch.

## 6. Recommended phased rollout

Mỗi phase = 1 PR. Sắp theo impact/effort + dependency.

### Phase 2A — P0 noindex + redirect cleanup (1 PR, low risk)
**Scope:** Fix I-7 + I-14 + I-15. Pure middleware + robots.txt.
- Add `/dang-ky/:token`, `/khoi-phuc-dang-ky` routes vào middleware → trả
  `renderNoindexShell` (giống `renderNotificationsShell` pattern).
- Add `/clb/:slug/quan-ly*`, `/social/:slug/{danh-sach,xep-cap,live}`,
  `/clubs/new`, `/clb/:slug/social/moi`, `/onboarding`, `/account`,
  `/login`, `/auth/*` → noindex shell.
- Update `public/robots.txt` để Disallow đúng những path private.
- Add `<meta robots="noindex">` vào `render404` (defense-in-depth).
- **Impact:** Hard-stop index pollution + magic-link privacy leak.
- **Effort:** Nhỏ, 1 file middleware + 1 helper + 1 robots.txt.
- **Risk:** None — chỉ thêm noindex, không xoá page.

### Phase 2B — P0 hub list prerender (1 PR, medium)
**Scope:** Fix I-1 + I-2.
- Add `renderSocialEventList(supabase, siteUrl, lang)` + `renderClubList`.
  Query 20 upcoming events + 20 clubs, emit ItemList JSON-LD, breadcrumb,
  proper title/desc/hreflang.
